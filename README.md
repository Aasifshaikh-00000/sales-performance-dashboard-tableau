# 📊 Sales Performance Dashboard Using Tableau

An interactive Sales Dashboard built in Tableau to visualize retail sales performance across products, customers, and locations. This project demonstrates my Tableau dashboard development, data visualization, and dashboard design skills using a multi-table retail dataset.

![Sales Dashboard](Images/dashboard.png)

---

## Project Overview

This project presents a comprehensive sales dashboard designed in Tableau. The dashboard combines sales, profit, and quantity metrics into a single interactive view, allowing users to explore business performance across different product categories and time periods.

The primary objective of this project was to strengthen my Tableau skills by designing a clean, interactive dashboard using multiple related datasets.

---

## Objectives

- Build an interactive dashboard in Tableau.
- Practice dashboard layout and visual storytelling.
- Display key business KPIs.
- Compare yearly sales performance.
- Visualize sales and profit trends.
- Analyze product sub-category performance.

---

## Dataset

The dashboard uses four related datasets:

| Dataset | Description |
|----------|-------------|
| Orders | Sales transaction details |
| Customers | Customer information |
| Products | Product details and categories |
| Location | Geographic information |

---

## Dashboard Features

### Executive KPIs

- Total Sales
- Total Profit
- Total Quantity
- Year-over-Year comparison

### Sales Analysis

- Monthly sales trend
- Highest and lowest performing months
- Weekly sales trend

### Profit Analysis

- Weekly profit trend
- Profit by product sub-category
- Loss-making product identification

### Product Performance

- Sales by sub-category
- Profit by sub-category
- Comparison with previous year
